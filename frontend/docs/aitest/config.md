# 配置中心

AI 模式配置、模型配置列表、提示词配置列表 UI 规范。

## AI 模式配置

### 页面布局

- 页面顶部：标题、保存按钮
- 主体区域：配置表单
- 支持两种模式：智能模式、专家模式

### 配置项

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| 模式选择 | 单选 | 智能模式 | 智能模式/专家模式 |
| 自动评审 | 开关 | 开启 | 是否启用自动评审 |
| 自动改进 | 开关 | 开启 | 是否启用自动改进 |
| 使用 LangGraph | 开关 | 关闭 | 是否使用 LangGraph 管线 |
| 并发数 | 输入框 | 1 | 并发执行数量 |
| 超时时间 | 输入框 | 300 | 超时时间（秒） |

## AI 模型配置

### 页面布局

- 页面顶部：标题、搜索框、新建按钮
- 主体区域：模型列表表格

### 模型表格

| 列 | 说明 |
|----|------|
| 模型名称 | 模型标识名称 |
| 提供商 | 模型提供商（阿里云/DeepSeek/OpenAI） |
| API 基础地址 | 模型 API 地址 |
| API Key | 掩码显示 |
| 状态 | 启用/禁用 |
| 创建时间 | 创建时间 |
| 操作 | 编辑、删除、设为默认 |

### 模型表单

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 模型名称 | 文本输入 | 是 | 模型标识 |
| 提供商 | 下拉选择 | 是 | 阿里云/DeepSeek/OpenAI |
| API 基础地址 | 文本输入 | 是 | API 地址 |
| API Key | 密码输入 | 是 | API 密钥 |
| 描述 | 文本域 | 否 | 模型描述 |
| 默认模型 | 开关 | 否 | 是否设为默认 |

## 提示词配置

### 页面布局

- 页面顶部：标题、搜索框、新建按钮
- 主体区域：提示词列表表格

### 提示词表格

| 列 | 说明 |
|----|------|
| 提示词名称 | 提示词标识 |
| 提示词类型 | 分析/生成/评审/改进 |
| 适用模型 | 关联的模型 |
| 创建时间 | 创建时间 |
| 操作 | 编辑、删除 |

### 提示词表单

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 提示词名称 | 文本输入 | 是 | 提示词标识 |
| 提示词类型 | 下拉选择 | 是 | 分析/生成/评审/改进 |
| 适用模型 | 多选 | 否 | 关联的模型 |
| 内容 | 文本域 | 是 | 提示词内容 |
| 描述 | 文本域 | 否 | 提示词描述 |

## API 接口

| Method | Path | Description |
|--------|------|-------------|
| GET | /api/v1/config/ai-mode | 获取 AI 模式配置 |
| PUT | /api/v1/config/ai-mode | 更新 AI 模式配置 |
| GET | /api/v1/config/models | 获取模型列表 |
| POST | /api/v1/config/models | 创建模型 |
| GET | /api/v1/config/models/{id} | 获取模型详情 |
| PUT | /api/v1/config/models/{id} | 更新模型 |
| DELETE | /api/v1/config/models/{id} | 删除模型 |
| GET | /api/v1/config/prompts | 获取提示词列表 |
| POST | /api/v1/config/prompts | 创建提示词 |
| GET | /api/v1/config/prompts/{id} | 获取提示词详情 |
| PUT | /api/v1/config/prompts/{id} | 更新提示词 |
| DELETE | /api/v1/config/prompts/{id} | 删除提示词 |

## 状态管理

使用 Pinia store 管理配置状态：

| Store | Description |
|-------|-------------|
| `ai_tester.ts` | AI 模式配置 |
| `configuration.ts` | 模型和提示词配置 |

## 配置优先级

1. 数据库配置（`AIModelConfig`）
2. 环境变量（`.env`）
3. 默认配置

API Key 解析链：
1. 优先使用数据库配置的 API Key
2. 若为空或含 `****` 掩码，回退到环境变量
3. 若两者都为空，使用默认 fallback

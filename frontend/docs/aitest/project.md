# 项目管理

项目列表、详情、成员管理 UI 规范。

## 项目列表页面

### 页面布局

- 页面顶部：标题、搜索框、新建按钮
- 主体区域：项目卡片列表或表格视图
- 支持列表视图和卡片视图切换

### 项目卡片

| 内容 | 说明 |
|------|------|
| 项目图标 | 项目类型图标 |
| 项目名称 | 项目全称 |
| 项目描述 | 简短描述，超出显示省略号 |
| 状态标签 | 进行中/已完成/暂停 |
| 统计信息 | 用例数、执行次数、通过率 |
| 操作按钮 | 查看详情、编辑、删除 |

### 项目表格

| 列 | 说明 |
|----|------|
| 名称 | 项目名称，可点击跳转 |
| 状态 | 状态标签 |
| 用例数 | 项目下测试用例数量 |
| 创建时间 | 项目创建时间 |
| 更新时间 | 最后更新时间 |
| 操作 | 查看、编辑、删除 |

## 项目详情页面

### 页面布局

- 页面顶部：项目名称、操作按钮（编辑、删除）
- 项目信息：描述、状态、创建时间、更新时间
- 统计卡片：用例数、执行次数、通过率、覆盖率
- 标签页：用例列表、执行记录、成员管理

### 成员管理

- 成员列表：用户名、角色、部门、加入时间
- 添加成员：弹窗选择用户，设置角色
- 角色权限：管理员、编辑者、查看者

## API 接口

| Method | Path | Description |
|--------|------|-------------|
| GET | /api/v1/projects | 获取项目列表 |
| POST | /api/v1/projects | 创建项目 |
| GET | /api/v1/projects/{id} | 获取项目详情 |
| PUT | /api/v1/projects/{id} | 更新项目 |
| DELETE | /api/v1/projects/{id} | 删除项目 |
| GET | /api/v1/projects/{id}/members | 获取成员列表 |
| POST | /api/v1/projects/{id}/members | 添加成员 |
| DELETE | /api/v1/projects/{id}/members/{user_id} | 删除成员 |

## 状态管理

使用 Pinia store `project.ts` 管理项目状态：

| State | Type | Description |
|-------|------|-------------|
| `projects` | Project[] | 项目列表 |
| `currentProject` | Project | 当前项目 |
| `members` | User[] | 成员列表 |
| `loading` | boolean | 加载状态 |

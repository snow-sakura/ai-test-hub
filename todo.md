## v0.1.0 - 2025-06-14

### P0 阶段：基础架构与 AI 核心

> 初始版本，完成全平台基础架构搭建和 AI 智能测试核心功能。

#### 新增功能

**基础架构**

- FastAPI 后端脚手架（SQLAlchemy 2.0 async + Alembic + Docker Compose）
- Vue 3 + TypeScript 前端脚手架（Vite 7 + Element Plus + Pinia）
- 暖色主题系统（#C67B5C）+ Layout 组件（侧边栏/顶栏/内容区）
- JWT 认证系统（登录/注册/Token 刷新/Topbar 集成）
- 首页仪表盘（统计卡片 + 9 功能模块入口网格）
- 国际化支持（中文简体 + 英文）

**AI 智能测试**

- AI 统一客户端（OpenAI/DeepSeek/Anthropic 多 Provider 支持 + 流式调用 + 续写）
- AI 用例生成（输入需求 → SSE 流式生成 → 表格展示）
- AI 评审（自动评审 + 根据评审意见改进）
- AI 评测师（综合评分 + 问题列表 + 修复建议）
- AI 测试报告（统计卡片 + 模块分布 + 用例明细）
- AI 智能模式配置（基础/模型/高级三个 Tab）

**配置中心**

- AI 模型配置管理（CRUD + API Key 脱敏）
- 提示词配置管理（CRUD + 内容预览）
- 生成行为配置（输出模式/自动评审/超时设置）

**系统管理**

- 用户管理（CRUD + 搜索 + 状态开关 + 重置密码）
- 角色权限（角色列表 + 权限配置）
- 系统设置（基本/安全/存储配置）
- 审计日志（多维筛选 + 操作记录）

#### 依赖

- Python 3.12+ | FastAPI 0.115+ | SQLAlchemy 2.0+ | aiomysql
- Node 18+ | Vue 3.5+ | Vite 7 | Element Plus 2.9+
- MySQL 8.0+ | Redis 7+

#### 文档

- `docs/plan/P0-基础架构与AI核心.md` — P0 阶段研发计划与架构说明
- `README.md` — 项目简介
- `backend/pyproject.toml` — Python 依赖清单
- `frontend/package.json` — 前端依赖清单
- `docs/database/数据库技术文档.md` — 数据库表结构完整文档（11 张表 + 中文注释）

#### 关键变更

- 新增后端源文件 29 个，前端源文件 70+ 个
- 50 个 API 端点
- 11 个 ORM 数据模型（11 张数据库表）
- 14 个前端页面组件
- 部署配置：docker-compose.yml + .env + Dockerfile
- 数据库 `ai_hub_test` 已创建，Alembic 初始迁移已完成
